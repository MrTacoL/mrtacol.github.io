# mrtacosi profile site


## Run locally

```bash
npm install
npm start
```

## Railway

Deploy this GitHub repo on Railway as a Node app. Railway should use:

```bash
npm start
```

## Music

The sound button is wired to this path:

```txt
assets/music.mp3
```

Upload your song file to `assets/` and it will play from the sound button.
